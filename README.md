Anti-Social Denver
==================

Aggregate by time of day and by lat/long "blocks"

An event can have an "area of effect"

### Time-based Events

* Crime
* 311
* Special events

### Day (6am to 2pm)

* Schools
* Restaurants
* Traffic
* Dispensaries
* Parks

### Evening (2pm to 10pm)

* Restaurants
* Bar / Clubs
* Traffic
* Parks 
* Dispensaries


### Night (10pm to 6am)

* Bar / Clubs

### Misc.

* Property sales
* foreclosures
